# 🍿 Poppy Albania Website

## Overview

Poppy Albania is a responsive promotional website developed using **HTML, CSS, JavaScript, and Bootstrap**. The website was created to showcase Poppy Albania's popcorn vending machines, providing visitors with information about the products, nutritional values, image galleries, and company details through a modern and user-friendly interface.

One of the main features of the project is bilingual support, allowing visitors to switch seamlessly between **Albanian** and **English**.

---

## Features

* 🌐 Responsive design for desktop, tablet, and mobile devices
* 🌍 Bilingual interface (English / Albanian)
* 🏠 Modern landing page with hero section
* 🍿 Product information and company overview
* 📊 Nutritional values table
* 🖼️ Interactive image gallery
* 📱 Mobile-friendly navigation menu
* ✨ Smooth scrolling and animations using Bootstrap and AOS
* 🔗 Social media integration

---
